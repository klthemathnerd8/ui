==Problem==

$\textbf{(A) } \frac{2}{3} \qquad\textbf{(B) } \frac{3}{10} \qquad\textbf{(C) } \frac{1}{2} \qquad\textbf{(D) } \frac{1}{3} \qquad\textbf{(E) } \frac{1}{4}$
